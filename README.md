# Wifi Speed Monitor

와이파이 연결 상태를 감시하고 실시간으로 다운로드 속도를 측정하여 화면에 표시해주는 백그라운드 유틸리티입니다.

## 주요 기능

- **자동 측정:** 프로그램 실행 시 및 와이파이 네트워크 변경 감지 시 자동으로 속도를 측정합니다.
- **단축키 지원:** 언제 어디서든 단축키를 눌러 현재 속도를 확인할 수 있습니다.
- **오버레이 알림:** 화면 우측 상단에 깔끔한 레드 컬러 텍스트로 측정 결과를 5초간 표시합니다.
- **백그라운드 실행:** 콘솔 창 없이 백그라운드에서 조용히 작동합니다.

## 실행 방법

1. `wifi_speed.exe` 파일을 다운로드합니다.
2. 파일을 더블클릭하여 실행하면 즉시 백그라운드에서 작동을 시작합니다.
3. (선택 사항) 시작 프로그램 폴더에 바로가기를 넣어두면, 컴퓨터를 켤 때마다 자동으로 실행됩니다.

## 단축키

- **`Ctrl + Shift + S`**: 현재 연결된 와이파이 속도 즉시 측정

## 참고 사항

- **측정 딜레이:** 속도 측정 로직 특성상 결과가 화면에 표시되기까지 **약 5초 내외의 시간**이 소요될 수 있습니다. 단축키 입력 후 잠시만 기다려 주세요.
- **윈도우 전용:** 본 프로그램은 Windows 환경의 `netsh` 명령어를 기반으로 제작되었습니다.
